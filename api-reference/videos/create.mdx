## Overview

Creates a new video project that will generate an AI-powered property tour video from a collection of images. The video project orchestrates multiple 5-second clips with smooth transitions and optional background music.

## Authentication

Requires an authenticated user session. The video project is created in the user's workspace.

## Request Body

<ParamField body="name" type="string" required>
  Name of the video project
  
  Example: `"123 Main St Property Tour"`
</ParamField>

<ParamField body="description" type="string">
  Optional description of the video project
</ParamField>

<ParamField body="aspectRatio" type="string" default="16:9">
  Video aspect ratio. Options: `16:9`, `9:16`, `1:1`
  
  - **16:9** - Landscape (YouTube, desktop)
  - **9:16** - Portrait (TikTok, Instagram Stories)
  - **1:1** - Square (Instagram Feed)
</ParamField>

<ParamField body="musicTrackId" type="string">
  ID of the background music track. Set to `null` for no music.
</ParamField>

<ParamField body="musicVolume" type="number" default="50">
  Music volume (0-100). Only applies if `musicTrackId` is set.
</ParamField>

<ParamField body="generateNativeAudio" type="boolean" default="true">
  Whether to generate ambient audio for clips using AI
</ParamField>

<ParamField body="clips" type="array" required>
  Array of clip configurations. Each clip represents a 5-second segment.
  
  <Expandable title="Clip object">
    <ParamField body="sourceImageUrl" type="string" required>
      URL of the source image for this clip
    </ParamField>
    
    <ParamField body="endImageUrl" type="string">
      Optional ending image for smooth transitions. Falls back to `sourceImageUrl` if not provided.
    </ParamField>
    
    <ParamField body="roomType" type="string" required>
      Room type for AI context. Options: `living-room`, `bedroom`, `kitchen`, `bathroom`, `exterior`, etc.
    </ParamField>
    
    <ParamField body="roomLabel" type="string">
      Custom label like "Master Bedroom" or "Front Yard"
    </ParamField>
    
    <ParamField body="sequenceOrder" type="number" required>
      Position in the video sequence (0-indexed)
    </ParamField>
    
    <ParamField body="motionPrompt" type="string">
      AI motion description. Example: "slow pan right, sunlight streaming through windows"
    </ParamField>
    
    <ParamField body="transitionType" type="string" default="seamless">
      Transition style. Options: `cut`, `seamless`
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique video project ID
</ResponseField>

<ResponseField name="workspaceId" type="string">
  Workspace ID
</ResponseField>

<ResponseField name="userId" type="string">
  Creator's user ID
</ResponseField>

<ResponseField name="name" type="string">
  Video project name
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `draft`, `generating`, `compiling`, `completed`, `failed`
</ResponseField>

<ResponseField name="clipCount" type="number">
  Total number of clips
</ResponseField>

<ResponseField name="estimatedCost" type="number">
  Estimated cost in cents. Formula: `clips × $0.35 × 100`
</ResponseField>

<ResponseField name="triggerRunId" type="string">
  Trigger.dev run ID for progress tracking
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp
</ResponseField>

## Next Steps

After creating the video project:

1. **Trigger generation** - Call the video generation endpoint to start processing
2. **Track progress** - Use the `triggerRunId` to monitor real-time progress
3. **Download video** - Once `status` is `completed`, download from `finalVideoUrl`

## Video Generation Pipeline

Once generation starts, the system:

1. **Generates clips** - Creates 5-second AI videos in parallel (up to 3 at a time)
2. **Creates transitions** - Generates seamless transitions between clips
3. **Compiles video** - Uses FFmpeg to merge clips with audio
4. **Uploads final video** - Stores in Supabase Storage

<Note>
Video generation takes approximately **2-5 minutes per clip**. A 10-clip video takes about 20-30 minutes total.
</Note>

## Cost Calculation

```typescript
// From source: lib/video/video-constants.ts
const CLIP_COST = 0.35; // USD per 5-second clip
const estimatedCost = clips.length × CLIP_COST × 100; // in cents
```

Example: 8 clips = 8 × $0.35 = $2.80 (280 cents)
